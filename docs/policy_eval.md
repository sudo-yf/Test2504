# Evaluation

## Real-time System Evaluation

```bash
uv run python main.py
```

- Press `q` or `ESC` to exit.
- On exit, system computes summary statistics and optional API-based trend analysis.

## Key Runtime Parameters

`config.yaml`:

- `emotion.detector_type`
- `emotion.detection_interval`
- `emotion.high_confidence_threshold`
- `data.database_path`
- `data.log_file`
