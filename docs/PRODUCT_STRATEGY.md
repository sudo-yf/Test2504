# 产品战略（EmotiSense）

## 1. 产品使命

构建一个“训练-检测-分析”闭环系统，帮助开发者快速落地可复现的情绪感知方案。

## 2. 目标用户

- 计算机视觉学习者与课程团队
- 需要快速验证情绪识别方案的开发者
- 需要轻量化训练与推理流水线的原型团队

## 3. 核心场景

- 场景 A：拉取公开数据，快速训练人脸检测模型
- 场景 B：微调情绪分类模型并在本地实时验证
- 场景 C：长期记录高置信度情绪日志并做趋势分析

## 4. 差异化定位

- 不做大而全平台，专注训练、实时检测、日志分析三条主线
- 在可复现工程链路上优先于单点算法脚本

## 5. 北极星指标（North Star Metric）

- 端到端实验闭环完成率（数据准备 -> 训练 -> 实时运行 -> 日志分析）

## 6. 关键指标

- 首次跑通时长（TTFV）
- 训练任务成功率
- 实时检测稳定帧率
- 日志完整率（高置信度事件入库成功率）

## 7. 产品路线

- V1（当前）：YOLO 训练 + Fine-tuning + 实时检测 + 日志分析
- V2：训练实验追踪（参数、指标、模型版本）
- V3：统一评估报告导出与可视化仪表板
